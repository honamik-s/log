About

過去の経験・学習・記憶が認知処理を促進するメカニズムに興味があります。

認知モデリングを使った情報処理過程の解明にも取り組んでいます。

主な研究テーマ

  • 時空間的統計情報の学習・利用メカニズムの解明
    • 文脈手がかり効果による注意処理促進メカニズム
    • 時間次元における確率情報の学習・利用メカニズム
    • 統計学習によって形成される記憶表象についての検討
  • 統計モデリングを用いた視覚的注意の空間特性の解明
  • 単語の並列処理に視覚的注意が及ぼす影響についての検討
